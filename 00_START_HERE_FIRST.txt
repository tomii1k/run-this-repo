## 📋 AFTER DEPLOYMENT (3 SIMPLE STEPS)

### Step 1: Configure Environment
```env
# Add to .env.local
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
```

### Step 2: Create Database Table
Run in Supabase SQL Editor (full SQL in README_DEPLOYMENT.md):
```sql
CREATE TABLE repo_analyses (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES auth.users(id),
  repo_url TEXT NOT NULL,
  analysis JSONB NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
);

ALTER TABLE repo_analyses ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Users can view their own" 
  ON repo_analyses FOR SELECT USING (auth.uid() = user_id);
-- ... (3 more policies)
```

### Step 3: Run & Test
```bash
npm run dev
```
Visit `http://localhost:3000/login`
