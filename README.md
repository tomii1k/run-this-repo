# RunThisRepo

AI-powered GitHub repository analyzer that generates setup guides, detects frameworks, explains environment variables, fixes common errors, and helps developers run open-source projects locally.

## Features

- Repository stack detection
- Beginner-friendly setup guides
- Environment variable analysis
- Common error explanations
- Security and safety checks
- Windows and macOS/Linux setup instructions

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- OpenAI API
- GitHub REST API
- Zod

## Goal

RunThisRepo helps developers and beginners quickly understand and run unfamiliar open-source projects without wasting time debugging setup issues.
